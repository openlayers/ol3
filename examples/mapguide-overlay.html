MapGuide dynamic overlay example

Example of dynamic overlay MapGuide map. Dynamic overlays are transparent and are suited for "mashing up" with other external layers. This example shows the Sheboygan map overlaid on top of the OSGeo WMS tile layer, which is in the same coordinate system

This example requires MapGuide Open Source 2.6 (installed on your own machine), which supports the new CREATERUNTIMEMAP operation

See the mapguide-overlay.js source to see how this is done.
