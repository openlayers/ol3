Custom overview map example

Example of a second smaller map defined and used to act as an overview map.

The overview map view's center is bound to the main map view's center, making both follow any center change. The overview map view's resolution being is also manually adjusted to 16 times the main map view's resolution using a listener on the 'change:resolution' event.

See the overviewmap-custom.js source to see how this is done.
