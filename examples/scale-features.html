Scale features example

Resizing features with the scale interaction.

This example demonstrates how the scale and select interactions can be used together. Zoom in to an area of interest and click to select a feature. Then drag one of the points that appears to scale the feature.

If you hold down Shift then the scaling will maintain the feature's aspect ratio.

See the scale-features.js source to see how this is done.
